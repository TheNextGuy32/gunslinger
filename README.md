# Gunslinger
######The Table Flipping Game
To play, visit www.not-in-service.com

###How to play
+ **A**, **D**: Move left and right
+ **W**: Flip or right tables (when close enough)
+ **S**: Crouch
+ **Left Mouse Button**: Fire gun
+ **K**: Reset game

###How to win
Clear the entire train of hostiles.
